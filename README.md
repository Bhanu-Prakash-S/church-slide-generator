# Church Slide Generator

A web app that turns section-marked lyrics and a performance order into a styled PDF of church song slides — in seconds.

Built to replace a 15–30 minute weekly Canva + iLovePDF workflow.

## Status

Work in progress. See [ROADMAP.md](ROADMAP.md) for milestones.

## Tech Stack

- **Backend:** Python, FastAPI, Pillow, img2pdf
- **Frontend:** React, TypeScript, Vite

## Quick Start

_Coming in Milestone 2 (backend) and Milestone 3 (frontend)._

## Why I Built This

Every week I prepare song slides for my church. The process: watch a YouTube video to learn the song order, open Canva, manually lay out each slide, export a 50–90 MB PDF, compress it on iLovePDF. This project automates all of that.
